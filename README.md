# AI-Powered Storage Tier Optimization System

## Overview

An intelligent storage optimization platform that classifies files into Hot, Warm, and Cold storage tiers using Machine Learning and provides archive recommendations to reduce storage costs.

## Features

* Storage Tier Prediction (Hot, Warm, Cold)
* Random Forest Classification
* K-Means Based Label Generation
* Priority Score Ranking
* Archive Recommendations
* Cost Savings Analysis
* Interactive Dashboard
* Cloud Deployment on Render

## Tech Stack

* Python
* Pandas
* Scikit-Learn
* Flask
* Plotly
* Bootstrap
* GitHub
* Render

## Workflow

Dataset Creation
→ K-Means Clustering
→ Storage Tier Labels
→ Random Forest Training
→ Priority Score Calculation
→ Flask Dashboard
→ Cloud Deployment

## Dashboard Features

* KPI Cards
* Storage Distribution Analysis
* Cost Analysis Charts
* Top Archive Candidates
* Smart File Analyzer

## Business Impact

The system helps organizations identify underutilized files, optimize storage allocation, and reduce storage costs through intelligent archiving recommendations.
